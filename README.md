# 🚀 ハッシュタグ生成ツール

文章を入力するだけで、最適なハッシュタグを自動生成する無料ツールです。

## ✨ 特徴

- 📝 文章から関連ハッシュタグを自動生成
- 🎯 ハッシュタグ数を調整可能（5〜30個）
- 📋 ワンクリックでコピー機能
- 🔥 人気ハッシュタグ一覧付き
- 📱 スマートフォン対応のレスポンシブデザイン
- 💰 広告収入モデル対応

## 🛠 技術スタック

- React 18 + TypeScript
- Tailwind CSS
- Vite
- デプロイ: Vercel

## 🚀 使い方

1. テキストエリアに文章を入力
2. ハッシュタグ数を選択
3. 「ハッシュタグ生成」ボタンをクリック
4. 生成されたハッシュタグをコピーしてSNS投稿に使用
